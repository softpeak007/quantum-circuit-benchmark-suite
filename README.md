# 🚀 QCBS – Quantum Circuit Benchmark Suite

An AI-powered open-source platform for quantum circuit benchmarking, fault-tolerant resource estimation, and reproducible Qiskit experiments.

## Features

- ✅ Automatic circuit generation
- ✅ Gate count analysis
- ✅ Circuit depth benchmarking
- ✅ Clifford+T resource estimation
- ✅ Native hardware metrics
- ✅ Experiment registry
- ✅ JSON & CSV logging
- ✅ Benchmark leaderboard
- ✅ Reproducible research pipeline

## Supported Circuits

- Bell State
- GHZ State
- Quantum Fourier Transform (QFT)
- Draper QFT Adder
- Ripple Carry Adder

## Technology Stack

- Python
- Qiskit
- Qiskit Aer
- Pandas
- Matplotlib

## Scientific Principles

- Reproducible benchmarking
- Transparent metric extraction
- Hardware-aware transpilation
- Fault-tolerant resource estimation
- No fabricated benchmark data
